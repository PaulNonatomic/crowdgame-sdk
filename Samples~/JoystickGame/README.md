# Joystick Game Sample

Movement-based game demonstrating joystick input and player spawning. Similar to the PoC soccer game.

## Features

- Virtual joystick input for player movement
- Player spawning and despawning on join/leave
- Physics-based Rigidbody movement
- Score tracking and display

## Setup

1. Import this sample via Package Manager
2. Open the JoystickGame scene
3. Press Play or use "Run Local Test" from the CrowdGame dashboard
