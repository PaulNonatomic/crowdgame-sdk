# Getting Started

## Installation

Add the CrowdGame SDK to your Unity project via the Package Manager:

```json
{
  "dependencies": {
    "com.nonatomic.crowdgame": "https://github.com/PaulNonatomic/crowdgame-sdk.git"
  }
}
```

## Quick Start

1. Open your Unity project (Unity 6, URP)
2. Add the CrowdGame SDK via Package Manager
3. Drag the **CrowdGame Platform** prefab into your scene
4. Subscribe to platform events in your game script
5. Click **Run Local Test** in the CrowdGame dashboard window
