# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - Unreleased

### Added
- Initial UPM package structure
- Core Platform API facade and service interfaces
- Player management system (PlayerSession, PlayerRegistry)
- Input system with control types (Joystick, Button, DPad, Swipe, Tilt, Shake, Text, Selection)
- Game lifecycle management (state machine with transitions)
- Messaging system (game-to-phone communication)
- Streaming service interface and configuration
- Local input provider for Editor simulation
- Editor dashboard window and PlatformConfig inspector
- CrowdGame Platform prefab
- JoystickGame sample
- Edit Mode unit tests
